# Mobideo

Mobideo is a small web application to record videos directly on a mobile device and upload
them to the server.

## Running

Mobideo is built with Sinatra and requires at least Ruby 2.2.

Run

    ruby app.rb

Then, goto http://localhost:4567 in your browser.
